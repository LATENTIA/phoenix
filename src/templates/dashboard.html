Add account

Each account gets its own folder under downloaded/ where you can drop historical CSV statements. The type determines tax handling (personal vs. business / ISOC).

Dividend data: in IBKR Client Portal, edit the Flex Query and add the Cash Transactions section, ticking Dividends, Payment In Lieu of Dividends, and Withholding Tax. Without it the current-year XML carries no dividend rows and the Dividends report only shows data from your annual CSV statements.

Add trade

Use this for trades that didn't come from IBKR (crypto exchanges, paper-trading corrections, family-account moves you want to track here). Manual rows are tagged with is_manual=1 in the database so they survive every re-ingest and can be removed via the API or by wiping the DB from the Settings page.
